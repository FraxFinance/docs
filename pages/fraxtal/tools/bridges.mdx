# Bridges

## Fraxtal OP Stack Bridge

The [Fraxtal Bridge](https://mainnet.frax.com/tools/bridge) allows you to bridge ETH, frxETH, and any other tokens from Ethereum to Fraxtal and vice versa.

## Fraxtal Ferry

The [Fraxtal Ferry](https://app.frax.finance/fraxferry) allows you to bridge FRAX, FXS, FPIS, FPI, sFRAX, and sfrxETH.

## Comparison

### Ethereum -> Fraxtal

|                  |  Fraxtal OP Stack Bridge  |         Fraxtal Ferry         |
| :--------------- | :-----------------------: | :---------------------------: |
| Supported tokens |            All            | Frax tokens, excluding frxETH |
| Duration         |        &lt; 1 hour        |           24 hours            |
| Delivery         | Automatically distributed |   Automatically distributed   |

### Fraxtal -> Ethereum

|                  | Fraxtal OP Stack Bridge |         Fraxtal Ferry         |
| :--------------- | :---------------------: | :---------------------------: |
| Supported tokens |           All           | Frax tokens, excluding frxETH |
| Duration         |         7 days          |           24 hours            |
| Delivery         |       Must claim        |   Automatically distributed   |
