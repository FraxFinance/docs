# Bridges

## Fraxtal OP Stack Bridge

The [Fraxtal Bridge](https://mainnet.frax.com/tools/bridge) allows you to bridge FRAX, and any other tokens from Ethereum to Fraxtal and vice versa.

## Frax Mesh

*For in-depth technical architecture and protocol design, see the official [Frax Crosschain Overview](https://docs.frax.com/protocol/crosschain/overview)*

Frax Mesh is the native, unified bridging infrastructure for all Frax ecosystem assets—including FRAX, frxUSD, sfrxUSD, frxETH, and sfrxETH—across multiple EVM-compatible chains such as Ethereum Mainnet, Fraxtal.

Users can initiate bridging directly within the Frax UI at https://frax.com/swap
